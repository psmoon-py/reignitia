# ReIgnitia

This project is a free, global, 3D mission-control SPA for students. It turns every part of student life—learning, finance, food, health, opportunities, curiosity—into personalized "missions", powered by free resources only.

---

## Tech Stack
- React + TypeScript
- Vite
- Tailwind CSS + shadcn/ui
- React Router DOM
- Framer Motion + GSAP (animations)
- three.js + @react-three/fiber + drei
- Zustand (state management)
- Local JSON for missions/resources

---

## App Routes
- `/`: Landing & Story
- `/launch`: Onboarding Wizard
- `/universe`: 3D Mission Control Constellation
- `/missions/list`: List View
- `/resources`: Resource Explorer
- `/opportunities`: Scholarships, Competitions
- `/account`: Profile & Settings
- `/about`: Credits & Backstory
