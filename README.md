# ReIgnitia

**Power Your Potential**

> No Student Left Behind. No Talent Wasted.

ReIgnitia is a global, free, student-first "mission control" designed to help students navigate their educational journey. Built for [Student Hackpad 2025](https://studenthackpad.devpost.com/), ReIgnitia provides integrated tools for learning roadmaps, scholarships, mental health, focus management, global STEM opportunities, and curated communities‚Äîall in one orbit.

## üöÄ What is ReIgnitia?

ReIgnitia (ree-ig-NISH-uh) is a comprehensive web platform that addresses the fragmented landscape of student resources. Instead of jumping between dozens of disconnected tools, students can access everything they need from a single, beautiful mission control interface.

### Core Philosophy

- **100% Free**: No paywalls, no subscriptions, no hidden costs
- **No Login Required**: Privacy-first, zero data collection
- **Client-Side Only**: All processing happens in your browser
- **Globally Accessible**: Works offline-first with cached data
- **Student-First**: Built by students, for students

## üåü The Six Labs

### 1. Learn Orbits
Interactive learning roadmaps for STEM fields using only free resources. Visualize your path from beginner to expert with curated steps from Khan Academy, MIT OCW, freeCodeCamp, and more.

### 2. Money Lab
Scholarship discovery engine and cost-of-study calculator. Find real opportunities from programs like INSPIRE, Google scholarships, DAAD, and Breakthrough Junior Challenge. Plan your education budget with realistic estimates.

### 3. Wellbeing Observatory
Micro mental health tools for stressed students. Quick breathing exercises, grounding techniques, and mood-based support‚Äîno therapy replacement, just practical first aid.

### 4. Time Focus Station
Study session structuring with research-backed focus presets. Pomodoro, exam sprints, gentle starts‚Äîall tracked locally to build your productivity patterns.

### 5. Opportunity Radar
Discover STEM competitions, hackathons, and fairs like NASA Space Apps, Regeneron ISEF, Breakthrough Junior Challenge, and regional opportunities you might have missed.

### 6. Peer Dock
Curated directory of existing free learning communities. Connect with OSSU CS, freeCodeCamp forums, Art of Problem Solving, Physics Stack Exchange, and more.

## üöÄ Getting Started

### Local Development

1. Clone this repository:
   ```bash
   git clone https://github.com/psmoon-py/ReIgnitia.git
   cd ReIgnitia
   git checkout static-pure
   ```

2. Serve with any static server:
   ```bash
   # Python
   python -m http.server 8000
   
   # Node.js
   npx serve
   
   # VS Code Live Server extension
   ```

3. Open `http://localhost:8000` in your browser

### Deployment

Deploy to any static hosting platform:

- **Netlify**: Drag & drop the folder
- **GitHub Pages**: Push to `gh-pages` branch
- **Vercel**: Import from GitHub
- **Cloudflare Pages**: Connect repository

All pages are static HTML‚Äîno build process required.
